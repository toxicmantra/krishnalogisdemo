3-Phase Project Plan

Step-by-step delivery plan — client decide kare kahan tak abhi jana hai

Phase 1 — Website Design & Development

Immediate · Core Deliverable

Goal: Ek modern, professional, premium logistics website banana — jo dekh ke client ko trust aaye aur company ka kaam clearly samjhe.

Included ✓

  • UI/UX Design — modern logistics industry standard
  • 5 Pages: Home, Services, Network/Coverage, About Us, Contact Us
  • Fully responsive — mobile, tablet, desktop
  • Company services detail — FTL, PTL, ODC, Door-to-Door, Textile
  • Office locations & addresses with Google Maps
  • Pan-India coverage section — RJ, MH, TN, GJ highlighted
  • Trust section — 700+ customers, 25 staff, min 50kg, ODC 150T capacity
  • Legal footer — CIN, registered company name, est. 2022
  • Basic SEO setup — meta titles, descriptions, page structure
  • Fast-loading, clean HTML/CSS/JS codebase
  • 1 round of revisions after first delivery

Not Included ✗

  • Revenue / financial figures anywhere on site
  • WhatsApp / email form backend integrations
  • Hosting or domain purchase
  • Live deployment to server
  • Track shipment / customer login portal
  • Content Management System (CMS) — static site in Phase 1

Deliverable: Complete website files (HTML/CSS/JS) ready for deployment.

Estimated Timeline: 10 – 14 working days after content & assets received.

Phase 2 — Hosting, Domain & Deployment

After Phase 1 · Engineer Coordination

Goal: Website ko live karna — client ke software engineer ke saath coordinate karke hosting, domain aur deployment setup karna.

Included ✓

  • Coordination with client's software engineer for hosting requirements
  • Domain configuration (e.g. techroute.co.in or new domain)
  • SSL certificate setup (HTTPS)
  • Website deployment to client's server / hosting
  • DNS configuration assistance
  • Post-deployment testing — all pages, mobile, speed check
  • Basic handover documentation

Not Included ✗

  • Hosting plan cost (client bears hosting charges)
  • Domain registration cost (client bears domain charges)
  • Ongoing server maintenance (separate AMC if needed)
Note: Phase 2 tab start hoga jab client ka software engineer contact kare aur hosting details share kare (server type, credentials, domain preference).

Deliverable: Live website accessible on client's domain with HTTPS.

Estimated Timeline: 3 – 5 working days after hosting details received.

Phase 3 — Integrations & Outreach Features

Future · After Go-Live

Goal: Website se directly customer reach-out enable karna — WhatsApp, email, quote forms aur advanced features.

Included ✓

  • WhatsApp click-to-chat button (floating + page CTAs)
  • Contact form → email delivery integration
  • "Get a Quote" form — From, To, Weight, Cargo Type, Phone
  • Form submissions sent to client's email
  • Click-to-call buttons on mobile
  • Google Analytics setup (visitor tracking)
  • Branch Locator — interactive state/city filter (if data provided)

Optional Add-ons (Phase 3+)

  • Shipment tracking page (requires backend API from client)
  • Customer login portal
  • CMS — client khud content update kare
  • Multi-language (Hindi + English)
  • Live chat integration

Deliverable: Fully interactive website with outreach integrations live.

Estimated Timeline: 7 – 10 working days after Phase 2 go-live.
